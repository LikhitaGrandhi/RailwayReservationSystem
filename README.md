# Railway Reservation System (Java + JavaFX)

A JavaFX-based railway ticket reservation system implementing *Confirmed, RAC, and Waiting List* logic similar to real railway booking systems.

## Features
- Ticket booking and cancellation
- Automatic RAC and Waiting List handling
- JavaFX graphical user interface
- File-based data storage
- Object-Oriented design

## Tech Stack
Java, JavaFX, Collections, File Handling
